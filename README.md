## Installation
1. Add the following line to your `Packages/manifest.json`:
```json
{
    "dependencies": {
        "com.pixel.meteoroid.ui.toolkit": "https://github.com/rafaelmfonseca/meteoroid-ui-toolkit.git",
    }
}
```
